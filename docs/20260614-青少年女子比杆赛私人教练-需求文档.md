# 青少年(女子)比杆赛 · 私人教练 App 需求文档(PRD)

> 日期:2026-06-14　状态:**完整需求稿(已整合此前分散草稿)**
> 定位:面向**青少年女子竞技选手**的**比杆赛**私人教练,覆盖**赛前→赛中→赛后→训练**完整闭环。
> 范围:**本期只做比杆赛(Stroke Play)这一条线**,做深做专;比洞/让杆/Stableford/排名赛列为远期扩展,本期不做。
> 主用户画像见 §1——产品所有默认值都为这个画像服务。
> 涉及具体数值处一律标 `〔待专业参数〕`,不预设。

---

## 0. 产品定位
把现有"记分 + 策略工具",做成一个**懂青少年女子选手的比杆赛私人教练**:像真人教练一样陪她**备赛—参赛—复盘—精进**。
衡量线:**一个好的青少年教练会做的,App 都要能做**。

## 1. 主用户画像(产品所有默认值的锚点)
- 15 岁,女子,竞技选手(目标:打好正式青少年比杆赛)。
- 比赛发球台 **5500–6200 yards**。
- **非力量型、挥速偏低 ⇒ 距离稳定、离散小**(核心优势:**可预测**)。
- 总距离不长 ⇒ 很多 par4 / par5 两杆 / 三杆**够不到果岭**。

**策略含义(贯穿全产品):**
① 吃"可预测"红利,把选杆 / layup / 攻旗算精,因为她的失误不会把精细计算抹掉;
② 用"**她自己的 par**"(见 §3)避免为抓官方 par 而爆洞。

## 2. 核心框架:私人教练全闭环
```
   【赛前】让她有底 → 【赛中】又记录又稳心 → 【赛后】数据说话 → 【训练】有计划攒小成功
        ↑                                                                      │
        └──────────────── 训练成果回流,能力模型一圈圈变准 ←────────────────────┘
```
| 阶段 | 教练做什么 | 关键产出 |
|---|---|---|
| 赛前 | 熟悉球场 + 个人化难度 + 重点练什么杆 | 整场基调 + 每洞计划(她的 par/保命洞/抓分洞) |
| 赛中 | 逐洞逐杆记录 + 实时建议 + **稳心**(冷静/聚焦当下一杆/失误引导) | 即时决策 + 情绪复位 + 总杆 |
| 赛后 | 五环节归因 + 同龄对照 | 得失分析 → 问题清单 → 训练重点 |
| 训练 | 短板→可打卡阶梯式处方,攒小成功 | 训练计划 + 能力模型更新 |

## 3. 贯穿全局的策略原则:**她自己的 par**
不照记分卡 par 抓分,而是按她真实能力给每洞定一个**战术 par**(例:某长 par4 当 par5 打——稳稳两杆到位、一切一推)。
全程核心:**别为了抓官方 par 去搏、搏出大数字**。比杆赛的灵魂 = **避免大数字**。

## 4. 阶段一 · 赛前(让她有底)
**4.1 熟悉球场**:每洞样貌 / 障碍 / 她那个台的码数 / 她大概怎么打(落点规划 + 她的 par)。支持**踩场(practice round)**预读。
**4.2 个人化难度**(不是球场 slope/rating,那是给差点的):以**她这套距离**判断"哪些洞对她难"——够不到果岭的洞 / 强制 carry / 窄球道 / 刁旗位。每洞标 **保命洞 vs 抓分洞**。`〔待专业参数:个人化难度算法〕`
**4.3 重点练什么杆**:从这个球场**反推**她会反复用到的距离段(如一堆 140–160 的第二杆)→ 赛前训练重点。赛前就把**球场与训练打通**。`〔待专业参数:反推规则〕`
**输出**:整场基调(偏稳/均衡,系统荐+可调) + 每洞计划(她的 par + 打法 + 保命/抓分标签)。

## 5. 阶段二 · 赛中(又记录、又稳心)〔运动心理是一等公民〕
**5.1 记录**:逐洞逐杆(沿用现有,补齐字段)。
**5.2 实时建议**:沿用已实现的逐杆建议(历史选杆 + 本轮手感 + 打法指引)。
**5.3 赛中心理引导 = App 扮演「一个好球童」**(基于真实球童实践,非杜撰;来源见 §13.2)
核心:赛中 App 不是冷工具,是她的**球童**——"战略家 + 情绪恒温器 + 信任的搭档"。青少年场景人设偏"鼓励 + 享受",非冷竞技。球童铁律(每条可直接做成 App 行为/话术):
1. **只活在当下一杆**:不显示总杆/名次,注意力永远指向"这一杆要做什么"。
2. **坏球固定复位**:失误后给平静、不评判、只指向下一杆的提示(可重复流程 > 临场意志);不复盘刚才那一杆的错。
3. **分清紧张 vs 不甘**:紧张(tension)要主动化解(降档/放慢/呼吸);适度不甘(irritability)别压抑,导向"下一杆打回来"(研究:引导得当下一洞反而更好)。失误话术按两类分化,不一律"别紧张"。
4. **好洞别飘**:打顺/连抓时把她拉回下一个开球的过程,别开始算"今天能打多少"。
5. **报码给全景 + good miss**:点明安全失误方向("别长,短了一天都没事"),让她有信念挥杆(现逐杆建议已部分在做,可强化 good-miss 一句)。
6. **护 pre-shot routine**:她专注的最后时刻不抛新信息,信息在她进入挥杆前给完。
7. **提醒她有多好 + 给视角**:低谷时"按平常打就行""一洞而已,路还长"。
8. **青少年特化:好球坏球都鼓励 + 确保享受**;家长当球童时先稳住自己情绪(会传染),并提示家长"该说/不该说什么"(含 Rule 10.2 给建议的边界)。

## 6. 阶段三 · 赛后(数据说话)
**6.1 五环节归因**:开球 / 长杆进攻 / 短杆 / 推杆 / 失误模式,看分丢在哪。`〔待专业参数:归因基准——strokes-gained 风格,但用青少年女子基准,非职业〕`
**6.2 同龄对照**:与**同年龄 / 同性别**群体比(数据来源正挖 **AJGA** 等,见 §8 硬骨头①)。
**6.3 输出**:得失分析 → 问题清单 → 直接生成训练重点。

## 7. 阶段四 · 训练(有计划、攒小成功)
- 训练处方做成**可量化、可打卡、阶梯式**:每个小目标都够得着,完成即标记、进步看得见。
- 复用 / 升级现有 AI 训练计划。
- **回流**:训练成果更新她的能力模型 → 下次赛前更准。

## 8. 支撑数据与三块硬骨头
**支撑模型**:
- 球员能力模型:各杆**近期**距离 / 离散 / 上果岭率 / 逐洞历史 / 状态,**带成长性、近期加权**(青少年几个月就长距离)。
- 球场模型:她台码数 / Par / 障碍 / 旗位 / 个人化难度。
- 同龄成绩库:同年龄 / 性别群体成绩。

**三块硬骨头(留给后续,需专业 + 真实数据,不靠猜):**
1. **同龄成绩库** —— 赛后对照的地基。来源**正在挖 AJGA**(进行中,结论见配套调研)。
2. **个人化难度算法** —— 把"球场 + 她的距离"映射成每洞难度与保命/抓分标签。
3. **赛中心理话术** —— 已重构为「球童角色模型」(§5.3),有真实球童实践 + 运动心理研究支撑(§13.2),框架成型;落地时具体话术细化建议再请运动心理专业人士校一遍。

## 9. 数据需求(新增需采集 / 录入)
- 球场:她台码数、障碍(水 / OB / dogleg / 强制 carry)、旗位、Stroke Index(可选)。
- 球员:年龄 / 性别、各杆**近期**距离与离散、逐洞历史、今天状态。
- 不采集:果岭精确坡度、风精确矢量。

## 10. 关键设计决策(已拍板)
| 决策 | 采用 | 理由 |

## 11. 分期
- **阶段一**:球员/球场模型补全(成长性 + 个人化难度) + 她的 par 策略 + 赛前。
- **阶段二**:赛中(记录 + 实时建议 + 稳心)。
- **阶段三**:赛后五环节归因 + 同龄对照(依赖硬骨头①)。
- **阶段四**:训练闭环 + 能力回流。

### 13.1 AJGA 同龄成绩库 · 调研结论(2026-06-14 实查)
**数据源**:AJGA 赛事成绩——2019 至今 `ajga.org/tournaments/archive`,2001–2018 `legacy.ajga.org`。实查样本:2018 EaglesDream Junior Championship(`tn=2018001`)。
**确认事实(一手,非推测):**
- 赛事 = 54 洞比杆赛,**Boys/Girls 分组独立排名**,年龄 12–19 → 同性别可切;"15 岁"需按选手年级/grad year 聚合子集。
- 成绩粒度 = **每轮 18 洞总杆 + to-par**(round-level),例:女子组 R1 领先 3-over-par 75。**无逐洞公开**。
- 球场 = **每场公开 Course Yardages**(可喂 §4.2 个人化难度的群体校准)。
- 平台 = 现代 Golf Genius,legacy 亦 JS/AJAX;完整榜单为**标签页 AJAX 加载**,规模化抓取需 CDP 或底层接口,纯静态拿不到。

**能支撑到(对 §6.2 同龄对照):**
- ✅ 同性别 × 球场难度 × **整轮成绩分布/百分位**("这种难度下同龄女子中位 79,你在第 X 百分位")。
- ✅ 球场码数 → 个人化难度群体基准。
- ⚠️ **做不到逐洞对照**(公开层无逐洞);逐洞失分归因靠她自己记录 + 通用基准。

**主源已定:美国 AJGA**(参赛体系确认走美国大学招募路线)。
**实施待办(归入阶段三,赛后对照):**
1. 抓取验证:用 CDP/Golf Genius 接口拉**一份完整女子榜单**,确认每选手每轮分 + **grad year/年龄字段可得**(决定"15 岁女子子集"能否成立)。
2. 采集管道:批量抓 2019–至今(+ legacy 2001–2018)女子组成绩 + 球场码数,聚合成"同龄女子 × 球场难度"分布。
3. 合规:公开赛事成绩,但守 robots/使用条款,控制抓取频率。

### 13.2 赛中心理「球童角色」· 调研来源(2026-06-14 实查)
支撑 §5.3 的"球童角色模型",均为真实来源(非杜撰):
- **U.S. Kids Golf 官方《Caddie Tips》**(L2 官方):赛前/中/后家长球童职责;赛中明确"好球坏球都鼓励、确保享受 + 及时帮做决策"。 `tournaments.uskidsgolf.com/.../parent-caddie-information/caddie-tips`
- **caddiehq《How to Caddy for a Professional Golfer》**(L3–L4):球童 = 战略家+心理师+情绪恒温器;坏球"平静接杆、只谈下一杆";报码全景 + good miss;pre-shot routine 神圣;好洞别飘。 `caddiehq.com/resources/how-to-caddy-for-a-professional-golfer`
- **claritycaddie《Sports Psychology for Golf》**(L4,引实证研究/β 系数):坏洞→紧张+烦躁滚雪球;tension 自我复合 vs irritability 可转专注;process focus;结构化 bounce-back protocol;差点越高滚雪球越狠。 `claritycaddie.com/blog/sports-psychology-for-golf`
- **The Caddie Network 真实 PGA 球童 FAQ**(L4 一手):坏轮拉回 = 提醒他多好 + 给视角(明天是新的一天)。 `thecaddienetwork.com/faq-how-do-you-get-your-player-back-into-it-mentally-when-theyve-had-a-bad-round/`
- **caddiehq《What to Say to Encourage a Golfer》**(L3–L4):分情境话术 playbook + **7 句绝不能说** + 按球员性格(分析型/情绪型/新手)调整。 `caddiehq.com/resources/what-to-say-to-encourage-a-golfer`

## 14. 附录A:青少年赛中心理话术库(可直接用;来源见 §13.2)
> 总原则(caddiehq):**少即是多,沉默是工具**;坏球后第一铁律——**不给技术建议**。

**A. 球童/家长对她说(外部鼓励,看时机):**
- 好球后(具体夸过程,别只"好球"):「这球真扎实」「方向压得准」「节奏很好」「这杆很果断」
- 坏球后(向前看、不复盘):「甩掉它,后面还长」「这洞翻篇了」「那球位运气不好,不怪你」「一次一杆,就看下一杆」「你能找回来」
- 难/高压杆前(少说,她紧张才给):「你可以的」「就柔和的节奏」「相信你的挥杆」「定了别犹豫」
- 洞间/等待(reset、别提刚才):「今天天气真好」「下面这洞你有机会」/ 聊点跟球无关的
- 青少年特化(Cohn):管理过高期望——「没人打得完美,早早一个坏球很正常,后面机会多」;**好球坏球都鼓励、确保享受**

**B. 她对自己说(self-talk / mantra,赛前选 2–3 条做"口袋卡"):**
- 自信:「我为这一刻准备好了」「我要打个果断的挥杆」「这杆我能打」
- 过程:「照我的 routine 做就够了」「让 routine 替我打这杆」「只打眼前这一杆」
- 失误 reset:「不管刚才怎样,每杆同一个流程」「一杆定义不了我,我的反应才定义我」「冷静+好奇:这球位和风要我怎么打?」
- 目标:「我想让这杆去哪?」(把"别打左"换成选定一个目标)

**C. 绝不能说的 7 句(家长尤其记,caddiehq):**
「头别抬」「放松点」「你刚才哪错了?」「你想太多了」「练习场你打得多好啊」「至少比我强」「要不试试改个动作」

**D. 两个可操作工具:**
- **情绪区(emotion zone,Dr.Mo)**:失误后允许她有"几步路"的情绪(想懊恼就懊恼),走完这几步就回 routine、翻篇——给青少年一个**有界的情绪出口**,而非压抑。
- **pre-shot + post-shot routine**:固定赛前流程 + 固定失误复位流程(Cohn:两个 routine 一起用,克服坏球)。

> 落地建议:A/C 可做成"给家长的小抄";B 可做成赛前自选的"口袋卡 mantra";D 做成失误后 App 引导的固定动作。具体上线话术建议再请运动心理专业人士校一遍措辞。
